# README — carpeta `analisis`

Esta carpeta reúne notebooks de análisis orbital y validación física para el estudio de Apophis.

## Contenido

- **`nb01_TransfHeliocentricaObservaciones.ipynb`**  
  Determinación orbital con incertidumbre a partir de observaciones.

- **`nb02_Cuadraturas_2BRP.ipynb`**  
  Cuadraturas y elementos orbitales en el problema relativo de dos cuerpos.

- **`nb03_Anomalias.ipynb`**  
  Cálculo e interpretación de anomalías orbitales para Apophis.

- **`nb4_kepler_benchmark.ipynb`**  
  Comparación de métodos iterativos para resolver la ecuación de Kepler.

- **`nb5_hodografo.ipynb`**  
  Estudio del hodógrafo y la geometría de la velocidad orbital.

- **`nb6_crtbp_Lpuntos_Cjacobi.ipynb`**  
  CRTBP, puntos de Lagrange y constante de Jacobi.

- **`nb7_perturbaciones_gauss.ipynb`**  
  Variación de elementos orbitales con ecuaciones de Gauss.

- **`observaciones_astrometricas.csv`**  
  Datos astrométricos usados en los análisis iniciales.

## Rol dentro del proyecto

`analisis/` complementa los modelos N-cuerpos con una capa interpretativa: explica fenómenos dinámicos, evalúa herramientas numéricas y conecta resultados con teoría orbital clásica.
